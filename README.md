## Quick Start

Install dependencies:

```console
 npm install
```

Start the server:

```console
 npm start
```

View the server at: http://localhost:4000

<br>

### Running Linters

To run the linters tool [Eslint](https://eslint.org/), first install the dependencies, then run:

```console
npm run lint
```

To fix the lint issues:

```console
npm run lint:fix
```
